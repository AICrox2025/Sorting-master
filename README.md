# Sorting Master 🎮

**Sorting Master** is an interactive terminal-based Rust game that helps you learn and practice sorting algorithms while having fun. Designed for both beginners and enthusiasts, it features levels, scoring, and a leaderboard.

---

## 🕹️ Features

- 🎯 **Interactive gameplay**: Sort numbers manually in the terminal.
- 🔀 **Algorithms**: QuickSort and BubbleSort.
- 🔥 **Level system**: Difficulty increases as you progress.
- 💀 **Hardcore mode**: Sort under a strict time limit (10 seconds).
- 🧾 **Leaderboard**: Keep track of high scores.
- 🎨 **ASCII UI with colors**: Enjoy a visually appealing terminal interface.
- ⏱️ **Score based on speed**: Faster sorting gives higher points.

## 📦 Installation

1. Install Rust (if you don’t have it):

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

2. Clone this repo

git clone https://github.com/AICrox2025/Sorting-master.git
cd Sorting-master

3. Build and run the game

cargo run
------------------------
**🎮 How to Play**
Enter your name when prompted.
Choose a game mode:
Normal – unlimited time
Hardcore – 10-second time limit
Choose a sorting algorithm:
QuickSort
BubbleSort
You will see a shuffled list of numbers.
Enter the numbers in sorted order (space-separated).
Points are awarded based on speed and correctness.
Advance through levels as you play.
Check the leaderboard to see the top scores.
**💡 Tips**
Use QuickSort for faster sorting of larger lists.
Try to beat your own high score!
Run in a terminal that supports ANSI colors for the best experience.
